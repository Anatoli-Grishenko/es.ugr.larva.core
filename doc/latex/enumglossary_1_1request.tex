\doxysubsection{Detailed Description}
Main commands that an agent can request in every session with the server. The syntax will be defined in a separate document 
\begin{DoxyItemize}
\item {\bfseries{login}} Open a session. All the members of the team can join it later. 
\item {\bfseries{join}} Connects to a previously open session. 
\item {\bfseries{logout}} Close a session 
\item {\bfseries{execute}} Ask for the execution of any of the agent\textquotesingle{}s capabilities 
\item {\bfseries{read}} Query the state of a sensor connected to the agent 
\end{DoxyItemize}
