\doxysubsection{Detailed Description}
\begin{DoxyAuthor}{Author}
Luis Castillo Vidal @ D\+BA It fixes Java\textquotesingle{}s VM grayscale bitmap issue regarding 3rd party grayscale imagers such as Gimp
\end{DoxyAuthor}
