\doxysubsection{Detailed Description}
A general-\/purpose class for logging messages, errors and exceptions, both on screen and on disk. It can be used from a general program or from an agent\textquotesingle{}s body. It admits as record a String or a Json\+Object and it produces the following log -\/ On screen\+: the string or the Json\+Object.\+to\+String() in the specified color -\/ On disk\+: every record on a single line on disk, encoded in J\+S\+ON, preceded by a timestamp -\/ \{\char`\"{}date\char`\"{}\+:\char`\"{}24/06/2020\+\_\+11\+:52\+:23\char`\"{},\char`\"{}record\char`\"{}\+:.....\}
