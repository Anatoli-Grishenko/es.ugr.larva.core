\doxysubsection{Detailed Description}
This is the basic agent in L\+A\+R\+VA. It extends a Jade Agent with an A\+PI of abstracted services with enhanced functionality or simplified operation regarding the core of Jade. These are the most important services 


\begin{DoxyItemize}
\item Simplified access to A\+MS 
\item Simplified access to DF with a pool of service agents and a pool of services provided. 
\item Logging capabilitiy both on screen and on disk with deactivable echo on screen. All the notifications are annotated with a timestamp and the own name of the agent in order to identify clearly who, when and what. It also differentiate three levels of information. For more information \mbox{\hyperlink{}{Logger}} 
\begin{DoxyItemize}
\item General information 
\item Error information 
\item Minor\+Exception handling 
\end{DoxyItemize}
\item Thanks to the use of Ole\+Passport, it offers automatic operation to load, store and transfer Passports. It supports encryption by the definition of an appropriate instance of Cryptor 
\item Support for reading, writing and transmission of any file, of any type and size. 
\item Provides a basic behaviour, which has to be acvivated nevertheless, in order to start working without any background on Jade behaviours. This is a repeatable behaviour (\mbox{\hyperlink{classagents_1_1LARVABaseAgent_a5e6d15a68ba1e7f3ddb3757985853c97}{Execute()}}) which acts as the main body of most agents and an associated boolean variable to control the exit and, therefore, the death of the agent. 
\end{DoxyItemize}
